Projects

Paths and Podcasts

Paths and Podcasts allows the user to select a starting location, a destination, and a podcast... and tells them how to travel based on the length of the episode. This wacky app was mob programmed with the great group of Russell Bentulan, Alissa Cheng, and Corey Derouin.

Made using React, REST APIs, JavaScript, HTML5, SCSS

Marsometer

This is an app that allows the user to input a city and compare its current temperature to that of Mars. It uses APIs from NASA InSight, OpenWeatherMap, and Teleport - and was done in collaboration with the wonderful and talented Bethan Davies.

Made using jQuery, REST APIs, JavaScript, HTML5, SCSS

Cash Comrade

Cash Comrade is a dynamic app that helps the user keep track of the money they want to save. One can enter the amount they want to reach, add or subtract money, and even estimate the time that it'll take to reach that goal.

Made using JavaScript, HTML5, SCSS
